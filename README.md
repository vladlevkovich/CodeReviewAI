## Installation

1. Clone:

   ```bash
   git clone https://github.com/vladlevkovich/CodeReviewAI.git
   cd repo```

2. Venv
```
python -m venv venv
source venv/bin/activate  # Linux / macOS
venv\Scripts\activate  # Windows
```
```
poetry install
```
